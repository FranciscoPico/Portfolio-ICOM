# ICOM5015 Artificial Intelligence Portfolio

This repository contains my portfolio for **ICOM5015 / CIIC5015 Artificial Intelligence** at the University of Puerto Rico at Mayagüez.

The portfolio includes my main coursework, programming assignments, and reflections on the course. It demonstrates my learning in AI topics such as search algorithms, constraint satisfaction problems, adversarial search, local search, and ethical issues in AI.

## Contents

* `Francisco_Pico_AI_Portfolio.pdf` — Final portfolio document
* `notebooks/` — Jupyter notebooks for AI assignments

## Featured Work

* **Brisca AI Project** — Game-playing AI using heuristic decision-making and adversarial reasoning.
* **Sudoku CSP Solver** — Sudoku modeled as a constraint satisfaction problem using variables, domains, and constraints.
* **Local Search Assignment** — Comparison of hill climbing, random restart, and simulated annealing.
* **AI Ethics Report** — Reflection on AI’s impact on employment, privacy, inequality, environment, and society.
